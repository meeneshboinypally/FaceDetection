# Face Detection and Recognition Using MTCNN and FaceNet

## Overview
This project implements real-time face detection and recognition using MTCNN for face detection and FaceNet for face embedding generation.

## Features
- Real-time face detection
- Face recognition using deep learning
- High accuracy face embeddings
- Easy to use and extend

## Technologies Used
- Python
- OpenCV
- MTCNN
- FaceNet
- NumPy
